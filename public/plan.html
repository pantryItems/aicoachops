ü§ñ AI Config Engine

How AI Builds the CRM Config

‚Ä∫
  1. Format intake answers into structured prompt with coaching context
  2. Call Claude Sonnet ‚Äî expert CRM architect persona, strict JSON schema output
  3. Parse & validate JSON config spec against Zod schema (retry once on parse failure)
  4. Store config in builds table: archetype, tags, custom fields, email templates, calendar config, branding
  5. Execute via GHL API ‚Äî create each item with step-by-step tracking and error handling
AI Output Includes: archetype classification, 8-15 niche tags, 5-10 custom fields, 3-5 email templates (personalized to coach voice), calendar configuration, branding guidelines, automation recommendations
